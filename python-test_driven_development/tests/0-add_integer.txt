The ``0-add_integer`` module
======================

Using ``add_integer``
-------------------

Test for the add_integer function.  First import
``add_integer`` from the ``0-add_integer`` module:

    >>> add_integer = __import__("0-add_integer").add_integer

Now use it:

    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98
    >>> add_integer(2)
    b must be an integer
    >>> add_integer(100.3, -2)
    a must be an integer
    >>> add_integer(4j, 2)
    a must be an integer
    >>> add_integer(2, 1**2)
    3
    >>> add_integer()
    a must be an integer
    >>> add_integer(100.3, "-2")
    a must be an integer
    >>> add_integer(100, "-2")
    b must be an integer
